{\bf Purpose}\\
Add a mobot to a mobot group.\\

\noindent
{\bf Return Value}\\
The function returns 0 on success and non-zero otherwise.\\

\noindent
{\bf Parameters}\\
A mobot handle attached to the mobot to add to the group.\\

\noindent
{\bf Description}\\
This function is used to add a mobot to a mobot group. \\
