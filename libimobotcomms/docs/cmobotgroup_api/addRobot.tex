{\bf Purpose}\\

\noindent
{\bf Return Value}\\
The function returns 0 on success and non-zero otherwise.\\

\noindent
{\bf Parameters}\\
A robot handle attached to the robot to add to the group.\\

\noindent
{\bf Description}\\
This function is used to add a robot to a robot group. \\
