{\LARGE \texttt{CMobot::motionTurnLeft()}\index{motionTurnLeft()}}\\
%\phantomsection
\addcontentsline{toc}{section}{motionTurnLeft()}

\noindent
{\bf Synopsis}\\
\begin{verbatim}
#include <mobot.h>
int CMobot::motionTurnLeft();
\end{verbatim}

\noindent
{\bf Purpose}\\
Rotate the MoBot using the faceplates as wheels.\\

\noindent
{\bf Return Value}\\
The function returns 0 on success and non-zero otherwise.\\

\noindent
{\bf Parameters}\\
None.\\

\noindent
{\bf Description}\\
This function causes the MoBot to rotate the faceplates in opposite directions
to cause the robot to rotate counter-clockwise.\\

\noindent
{\bf See Also}\\
\texttt{motionTurnRight()}
