The "chmobot" package is a Ch wrapper for the Barobo MoBot control library. The
package allows Ch scripts to run MoBot control programs in Ch script space,
without need for compilation.

Installation for Windows:
(1) Go into the directory created by unzipping the zip file with the command
    
    cd chmobot-0.1.0

(2) Type the command:

    make install

Installation for Mac and Linux:

(1) Go into the directory created by unzipping the zip file with the command
    
    cd chmobot-0.1.0

(2) Type the command:

    sudo make install
